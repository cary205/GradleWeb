# GradleWeb

## A Web project using Gradle, Tomcat, Struts...

## Install Gradle

    sudo add-apt-repository ppa:cwchien/gradle
    sudo apt-get update
    sudo apt-get install gradle

## Run project

    gradle clean tomcatRun
